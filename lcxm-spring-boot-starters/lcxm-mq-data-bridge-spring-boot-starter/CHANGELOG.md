### 2025-05
- [x] **指定消息生产、发送和消费标识**，一般用于开发环境，如果有特殊消费需求，可按需配置
  - 生产标识：data.bridge.produce-flag=xxx 
    - 产生的消息携带该标识
    - 默认为空，不携带
  - 发送标识：data.bridge.send-flag=xxx 
     - 发送数据库消息时候只发送该标识的消息
     - 如果不配置，则发送全部可发送的消息
  - 消费标识：data.bridge.consume-flag=xxx
    - 消费数据库消息时候只消费该标识的消息
    - 如果不配置，则消费全部可消费的消息
